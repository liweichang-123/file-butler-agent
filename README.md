# 📁 本地文件管家智能体

基于 AI 智能体的本地文件整理工具，自动扫描目录、分类文件、生成整理建议。

## 功能特性

- 🔍 **智能扫描**：自动扫描指定目录下的所有文件
- 📊 **自动分类**：按文件类型（图片、文档、视频等）自动分类
- 🤖 **AI 建议**：调用大模型生成智能整理建议
- 🔒 **安全只读**：仅生成建议，不会实际移动或修改任何文件
- 🌐 **Web 界面**：基于 Gradio 的友好交互界面

## 技术栈

- Python 3.11
- Ollama + Qwen2.5:7b（本地大模型）
- Gradio（Web 界面）
- pytest（自动化测试）
